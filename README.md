# [Examples for ``dyce`` and ``anydyce``](https://posita.github.io/dyce-notebooks)

 This repository provides [interactive Jupyter notebooks to showcase ``dyce`` and ``anydyce``](https://posita.github.io/dyce-notebooks).
 ``dyce`` offers [simple Python tools for exploring dice outcomes and other finite discrete probabilities](https://posita.github.io/dyce).
 ``anydyce`` [offers visualization tools for ``dyce``](https://posita.github.io/dyce).

Try them out for yourself: https://posita.github.io/dyce-notebooks
